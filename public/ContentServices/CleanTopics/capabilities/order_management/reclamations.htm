"- Reclamations present a sub-set of order items - A new order could be created out of a reclamation does the reclamation involve a refund? or return of the original purchase? In CORE - no, it does not involve a refund. But on the project level, it can be easily done and reclamation button will be connected to refund. Reclamation was released this way because of the dependencies on state machine. We cant command what state machine must be used in projects. Refund - one of the states of state machine."
